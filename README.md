## ✨ Features

| Feature | Description |
|---|---|
| 🌓 **Dark / Light Mode** | Toggle between themes with smooth transitions |
| 🔍 **Live Search** | Instant menu filtering as you type |
| 🛒 **Smart Cart** | Add, remove, update quantities with live bill calculation |
| 🎟️ **Promo Codes** | Apply discount coupons at checkout |
| 🏷️ **Menu Filters** | Filter by Veg, Non-Veg, Bestseller, Spicy, and more |
| 📊 **Sort Options** | Sort menu items by price (asc/desc) or rating |
| ❤️ **Wishlist** | Save favourite dishes for later |
| 📋 **Order History** | View past orders and reorder with one tap |
| 📦 **Order Tracker** | Real-time order status progress bar |
| 📍 **Address Manager** | Switch between saved delivery addresses |
| 👤 **Login / Register** | Email, Google & OTP sign-in support |
| 🎉 **Order Success Screen** | Animated confirmation with order number |
| 📱 **Fully Responsive** | Works perfectly on mobile, tablet & desktop |
| 🍔 **Item Detail Modal** | Full description, customisation options & add-ons |
| ⭐ **Customer Reviews** | Star ratings and review cards section |
| 📣 **Offer Strip** | Auto-scrolling live offer ticker at top |

## 🚀 Getting Started

### Option 1 — Just Open It
```bash
# Clone the repo
git clone https://github.com/shubhamnarware67-cmd/bgi-food-bhopal.git

# Navigate to the folder
cd bgi-food-bhopal

# Open directly in your browser
open bgi_food_bhopal.html
```

### Option 2 — Live Server (VS Code)
1. Install the **Live Server** extension in VS Code
2. Right-click `bgi_food_bhopal.html`
3. Select **"Open with Live Server"**

### Option 3 — GitHub Pages
Deploy for free on GitHub Pages:
```
Settings → Pages → Source: main branch → /root → Save
```
Your app will be live at: `https://shubhamnarware67-cmd.github.io/bgi-food-bhopal`

## 🎨 Design Highlights

- **Color Palette** — Signature BGI Orange (`#FF5722`) with deep dark hero section
- **Typography** — Syne (bold headings) + DM Sans (clean body text)
- **Animations** — Smooth `fadeUp`, `pulse`, `pop`, `float`, and scroll ticker
- **CSS Variables** — Full dark/light theming via `:root` and `[data-theme=dark]`
- **Glassmorphism** — Backdrop blur cards in the hero food mosaic
